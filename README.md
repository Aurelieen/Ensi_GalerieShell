# Ensi_GalerieShell
Ensimag 1A – Génération d'une galerie d'images en HTML à partir de scripts shell

## Installation
```sh
git clone https://github.com/Aurelieen/Ensi_GalerieShell.git
```

## Usage
```sh
# Lancer l'utilitaire
./galerie-shell.sh -h
```
```sh
# Accéder à la liste des tests
firefox rapport/index.html
```
